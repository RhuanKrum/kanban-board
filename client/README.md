# Kanbam Board Client

Simple kanbam board application using NodeJS and React, and using mainly socket.io for real-time communication between client and server.

Image Here!

## Client

Using the following tools:
* bootstrap
* bootstrap-icons
* query-string
* react
* react-bootstrap
* react-dom
* react-emoji
* react-router
* react-router-dom
* react-scripts
* react-scroll-to-bottom
* socket.io-client

## Server

Using the following tools:
* cors
* eslint
* eslint-plugin-node
* express
* nodemon
* pg
* pg-hstore
* sequelize
* socket.io
